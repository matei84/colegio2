Cosas de github: https://git-scm.com/book/es/v2/Fundamentos-de-Git-Trabajar-con-Remotos

Para descargar las cosas que he realizado de github: git pull colegio2 main (el primer termino para conocer desde que repositorio se descarga la info y el segundo termino para saber desde que branch de ese repositorio).

Para subir las cosas que he realizado desde local: git push colegio2 main (el primer termino para conocer hacia que repositorio se sube la info y el segundo termino para saber a que branch de ese repositorio va la info).

Para saber cuales son mis conexiones remotas de github: git remote -v

Para eliminar github remoto: git remote rm origin (donde origin es desde donde me conecto del remoto)
